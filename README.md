# CSV2XLSX

CSV2XLSX is a lightweight Windows tool for converting CSV files into Excel `.xlsx` workbooks.

## Features

- Convert CSV files to `.xlsx`
- Drag & drop CSV files onto the main window
- Batch conversion of multiple CSV files
- Automatic CSV delimiter detection
- Manual delimiter selection if detection is ambiguous
- Column selection before export
- Live data preview before conversion
- Saves selected window sizes
- Resizable modern UI
- Optional “Always on top”
- Optional minimize to system tray
- Portable single-exe publishing supported
- Uses embedded application icon and resources
- Built with WinForms, CsvHelper and ClosedXML

## Requirements

- Windows
- .NET 8 Desktop Runtime
